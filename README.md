# AND-Coding-Challenge_Langton-Ant
Challenge 2 - "Antics"

---

<b>Requirements</b>

1. Java >= 11

---

<b>Controls (Ant)</b>

1. <b>LEFT ARROW</b> = Slow Down
2. <b>RIGHT ARROW</b> = Speed Up
3. <b>T</b> = Turbo Mode :)
4. <b>F</b> = Follow the Ant

---

<b>Controls (Other)</b>

1. <b>BACKSPACE</b> = Restart
2. <b>ESCAPE</b> = Exit

---

Framework: 
  - LibGDX -> https://github.com/libgdx/libgdx
